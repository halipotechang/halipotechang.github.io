CSS 文字美化

CSS 文字美化


字體美化


  • font-family 用來設定字體類別

  • font-size 用來設定字體大小

  • font-weight 用來設定字體厚度

  • font-style 用來設定字體是否為斜體

參數： italic、oblique

  • font-variant 設定是不是要以小型大寫字體顯示

參數： small-caps、normal


文字方向


  • direction 設定文字起始位子

參數： ltr(左到右) 、 rtl(右到左)


文字字距


  • letter-spacing 設定字母的間距

文字行高


  • line-height 設定每一行之間的距離

文字對齊


  • text-align 設定文字是如何對齊

參數： left 、 right 、 center 、 justify(左右對齊)


文字裝飾


  • text-decoration 設定文字上有什麼裝飾

    • underline : 文字下加上一條線

    • overline : 文字上加上一條線

    • line-through : 文字中穿越一條線


文字縮排


  • text-indent 設定每ㄧ段的首行縮排效果

文字變形


  • text-transfrom 控制大小寫是如何展現

    • capitalize: 每一個字的第一個字母以大寫顯現

    • uppercase : 所有字母以大寫展現

    • lowercase : 所有字母以小寫展現


單字字距


  • word-spacing 設定每ㄧ個英文單字的距離
