JAVA 運算子

JAVA 運算子

算術運算子


算術運算子 說明
+ (+ 數字 加法運算) (+ " " 字串串接)
- 減法
* 乘法
/ 除法
% 取餘數

遞增遞減運算子


遞增遞減運算子 說明
a++ a = a + 1
a– a = a - 1

  • 後置型 (a++):先取值後再遞增

  • 前置型 (++a):先遞增後再取值


指定運算子


將右邊運算完成的結果指定給左邊的變數

指定運算子 用法 說明
= a = 1 a = 1
+= a += 1 a = a + 1
-= a -= 1 a = a - 1
*= a *= 1 a = a * 1
/= a /= 1 a = a / 1
%= a %= 1 a = a % 1

關係運算子


關係運算子(Relational operators)用來比較兩個變數的值，其回傳值結果是一個布林值

關係運算子 說明
< 小於
<= 小於等於
== 等於
!= 不等於
> 大於
>= 大於等於

條件運算子


條件運算子 (Conditional operators)是將兩個布林值合併起來的運算子，運算結果仍為布林值

條件運算子 說明
&& and
|| or
! not

三元運算子


三元運算子 (Ternary operators)是一個簡略的if – else敘述

三元運算子 說明
a ? b : c (true = b) 、 ( false = c )
