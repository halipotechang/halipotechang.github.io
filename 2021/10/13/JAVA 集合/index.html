JAVA 集合

JAVA 集合

介紹

  • 集合物件是指一群相關聯的資料，集合在一起組成一個物件，存在裡面的資料，稱之為元素(element)

  • 從JDK1.2開始，Java的集合物介面分作兩大類:

    1. Collection介面:可持有各自獨立的物件
    2. Map介面:持有成對的key-value物件
